mazeNavigatorGameJs

Hello, this is a fun liitle project by Pranit, made with Javascript.The game is best played in desktop or laptop

This is a simple maze game where you need solve the maze by moving a ball through a maze to its goal which is a green box.

Controls:
Use "W" key for moving the ball up
Use "A" key for moving the ball left
Use "S" key for moving the ball down
Use "A" key for moving the ball right
